# UAS

(penjelasan singkat proyek)
Program ini adalah website kuis pilihan ganda yang:
Menampilkan 5 soal: 
Setiap soal punya 4 pilihan jawaban: 
Pengguna harus menjawab semua soal Setiap jawaban benar bernilai 20 poin Total skor maksimal = 100 Jika skor > 70 → LULUS, jika tidak → TIDAK LULUS Ada tombol Submit dan Ulangi Kuis Struktur program dibagi 3: index.html → menampilkan soal & tampilan web style.css → mengatur tampilan (warna, layout, dll) script.js → mengatur logika kuis & perhitungan nilai

(alur logika/algoritma)
Mulai Set skor = 0 Untuk setiap soal 1 sampai 5: 
Jika belum dijawab: 
Tampilkan pesan "Semua soal harus dijawab" stop Jika dijawab:
Tambahkan nilai jawaban ke skor Jika skor > 70: 
Tampilkan "LULUS" Else:tampilkan "TIDAK LULUS" Selesai
